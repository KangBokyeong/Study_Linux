# [Linux Journey 2] Text-Fu
## stdput(Standard Out)
- 아직 정리중... (공부중이라 정리는 나중에 꼭 할 것!!)
- 파일 출력
- 텍스트에 글자 입력 및 택스트 출력

|명령어|기능|
|:-----:|:-----:|
|echo Hello World **>** peanuts.txt |peanuts라는 텍스트 파일을 만들어 Hello World 입력|
|echo Hello World **>>** peanuts.txt |peanuts라는 텍스트 파일에 Hello World 입력(파일이 이미 존재 할 때)|
