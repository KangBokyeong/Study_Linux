# [Linux Journey 3] Advanced Text-Fu
## 1. regex (Regular Expressions)
- 패턴 기반 선택을 수행하는 강력한 도구
- 특수 표기법 사용
- ^로 라인 시작
- $로 라인 마침
- .으로 모든 단일 문자 일치
- []로 대괄호 표기법
  - 대괄호 안에있는 문자 지정 가능 
  - 범위를 사용하여 사용하려는 문자의 양 늘릴 수 있음.
  - 주의: 대소문자를 구분함.
  
## 2. Text Editors
- Vim과 emacs는 대부분의 Linux 배포판에 기본적으로 설치되는 인기있는 텍스트 편집기
- ninja처럼 시스템을 돌아 다니고 싶다면이 텍스트 편집기 중 하나를 선택하여 사용해야 함.

## 3. Vim (Vi Improved)
- vi 텍스트 편집기 명령의 향상된 버전을 나타내는 이름과 마찬가지로 vi (Improved)의 약자.
- 임의의 Linux 배포판을 부팅한 경우 vim 이 기본적으로 설치됨.

## 4. Vim Search Patterns
- / : / 뒤에 검색 결과 입력 후 Enter를 누름.
  - 검색 결과에서 "n"을 누르면 앞으로 이동
  - 검색 결과에서 "N"을 누르면 뒤로 이동
- ? : 탐색 명령어
  -  택스트 파일을 거꾸로 검색

## 5. Vim Navigation
- 여기서는 마우스를 사용할 수 없음. 
- 그래서 아래의 조작키를 이용해야 함.(명령 모드)

|명령어|기능|
|:------:|:------:|
|h 또는 왼쪽 방향키|왼쪽으로 한 글자 이동|
|k 또는 위쪽 방향키|위쪽으로 한 줄 이동|
|j 또는 아래쪽 방향키|아래쪽으로 한 줄 이동|
|l 또는 오른쪽 방향키|오른쪽으로 한 글자 이동|

## 6. Vim Appending Text
- 텍스트 삽입을 위한 삽입 모드

|명령어|기능|
|:------:|:------:|
|i|커서 앞에 텍스트 삽입|
|O|이전 행에 텍스트 삽입|
|o|다음 줄에 텍스트 삽입|
|a|커서 뒤에 텍스트 추가|
|A|줄 끝에 텍스트 추가|

- 삽입 모드 종료 시 Esc 키를 눌러주기

## 7. Vim Editing
- 편집 및 일부 크래프트 제거

|조작키|기능|
|:------:|:------:|
|x|선택한 텍스트 자르기, 문자 삭제|
|dd|현재 행 삭제|
|y|선택한 항목을 뽑거나 복사|
|yy|현재 줄을 채우거나 복사|
|p|복사 한 텍스트를 커서 앞에 붙여 넣기|

## 8. Vim Saving and Exiting
- 저장 및 종료

|조작키|기능|
|:------:|:------:|
|w|파일을 쓰거나 저장|
|q|vim 종료|
|wq|쓰고 종료|
|q!|저장하지 않고 vim 종료|
|ZZ|wq와 동등하지만 1문자 더 빠름|
|u|마지막 작업 취소|
|Ctrl-r|마지막 작업 다시 실행|

## 9. Emacs
- 모든 코드 편집, 파일 조작 등을 모두 할 수 있음.
- 로드 속도가 약간 느려지고 학습 곡선이 vim 보다는 약간 가파름.
- 확장성이 뛰어남.
- emacs의 버퍼: 텍스트가 있는 파일 => 파일을 열명 해당 파일의 내용을 저장하는데 버퍼가 사용됨.
- 동시에 여러 개의 버퍼를 열어서 버퍼 간 쉽게 전환 가능.
 
## 10. Emacs Manipulate Files
### 파일 저장
- C는 Ctrl를 의미함.

|조작키|기능|
|:------:|:------:|
|C-x C-s|파일 저장|
|C-x C-w|~로 파일 저장|
|C-x s|모두 저장|

- 파일 저장 옵션: 각 파일을 저장할지 묻는 메시지 표시

### 파일 열기
- 열 파일 이름을 입력하는 메시지가 나타남.
- 이미 존재하는 파일이 없는 경우, 새 파일 생성.
- 디렉토리도 로드 가능.

|조작키|기능|
|:------:|:------:|
|C-x C-f|파일 열기|

## 11. Emacs Buffer Navigation
- 버퍼 (혹은 방문한 파일)를 이동하기 위해 아래 명령어 사용.
### 스위치 버퍼

|조작키|기능|
|:------:|:------:|
|C-x b|스위치 버퍼|
|C-x right arrow|버퍼를 통한 오른쪽 사이클|
|C-x left arrow|버퍼를 통한 왼쪽 사이클|

### 버퍼 닫기

|조작키|기능|
|:------:|:------:|
|C-x k|버퍼 닫음|

### 현재 버퍼 분할
- 한 화면에서 여러 버퍼 볼 수 있음.

|조작키|기능|
|:------:|:------:|
|C-x 2|현재 버퍼를 분할함.|

### 단일 버퍼를 현재 화면으로 설정

|조작키|기능|
|:------:|:------:|
|C-x 1|단일 버퍼를 현재 화면으로 설정함.|

## 12. Emacs Editing
### Text Navigation
- 텍스트 탐색

|조작키|기능|
|:------:|:------:|
|C-up arrow|한 단락 위로 이동|
|C-down arrow|한 단락 아래로 이동|
|C-left arrow|한 단어 왼쪽으로 이동|
|C-right arrow|한 단어 오른쪽으로 이동|
|M->|버퍼의 끝으로 이동|

### Cutting and Pasting
- 자르기 및 붙여놓기

|조작키|기능|
|:------:|:------:|
|C-space key|텍스트 선택|
|C-w|자르기|
|C-y|붙여놓기|


## 13. Emacs Exiting and Help
### 종료
- 열려있는 버퍼가 존재하면, emacs가 닫히기 전에 저장하라는 메시지가 표시됨.

|조작키|기능|
|:------:|:------:|
|C-x C-c|emacs 종료|

### 헷갈릴 경우

|조작키|기능|
|:------:|:------:|
|C-h C-h|도움말 메뉴 표시|

### 되돌리기

|조작키|기능|
|:------:|:------:|
|C-x u|되돌리기|
